This is my first game that I'm actually selling on steam! This is so far my best game dev project lolol

This is a multiplayer coop game (with max 8 people) where you and your team defend the CORE against waves of enemies based on aesthetics (weirdcore, dreamcore, webcore, angelcore, etc.)




If you want to read more about the game, check out the steam page.


Michael's Description

i actually wanted to make this project during 8th grade summer but wasn't good enough at coding

i am a giant fan of aesthetics, as i've often had phases where i'd get really obsessed with liminal spaces and these aesthetics

so during my first semester of college, i tried again and so far it's going so well!

i figured out multiplayer which was really hard to do; lowkey this game could pay off my entire undergrad so im LOCKED IN

i need around 25k-30k wishlists. so far (3/5/26) i am at around 2k. progress for marketing is slow but we will get there, trust.
